# nvm-fish

Fish shell integration for Node Version Manager (nvm) on Arch Linux.

## 概述

这个包为 Fish shell 提供了 nvm (Node Version Manager) 的集成支持。

## 依赖

- `nvm` - Node Version Manager（Arch Linux extra 仓库）
- `fish` - Fish shell
- `bass` - Bash 到 Fish 的转换工具

## 安装

从 AUR 安装：

```bash
yay -S nvm-fish
```
